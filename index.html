安心のGoogle連携

Tour Riseでは、Gmail・スプレッドシート・カレンダーとの連携に、GoogleのOAuth認証を利用しています。 お客様のGoogleアカウントと連携する際には、明確な許可をいただいた上で必要最小限のデータのみを扱い、 業務効率とセキュリティを両立しています。

Googleアカウント連携について

Tour Riseは以下のGoogle APIを利用して、予約管理を自動化します：

  • Gmail API

    予約メールの読み取りのみに使用（メール送信や削除は行いません）

  • Google Sheets API

    予約情報の記録や管理のために使用

  • Google Calendar API

    確定した予約をカレンダーに登録するために使用

これらのAPIへのアクセス権限をいただくことで、手動作業を自動化し、業務効率を大幅に改善します。お客様のデータは予約管理の目的以外には使用せず、厳重に保護します。

よくある質問

Q: 既存のGoogleアカウントで使えますか？

A: はい、Google Workspaceを契約していない無料のGoogleアカウントでもご利用いただけます。

Q: 対応している予約プラットフォームは？

A: TripAdvisor、Klook、GetYourGuideなど主要な予約プラットフォームに対応しています。その他のプラットフォームについても、多くの場合対応可能です。詳細はお問い合わせください。

Q: 解約するとデータはどうなりますか？

A: 解約時にはシステムへのアクセスが停止されますが、それまでに蓄積されたデータはGoogleスプレッドシート上に残ります。
